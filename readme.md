# Project Title

Build a Cryptocurrency Price Tracker

---
## Requirements

For development, you will only need Node.js and a node global package, installed in your environement.

---
## Install

    $ git clone https://github.com/goutham41/Monitor-Bitcoin.git
    $ cd server
    $ npm install
    
---
## Running the project

    $ node index.js 
    $ (or) npm run start
